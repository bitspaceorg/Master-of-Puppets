Master of Puppets is a backend-agnostic viewport rendering engine written in C11. It provides a clean, layered architecture for rendering 3D geometry through interchangeable backends — CPU software rasterization, OpenGL, or Vulkan — behind a single public API.

## Start Here

**[Skill](skill)** — self-contained, task-oriented entry point. Build + render loop, all common API with signatures, capability matrix per backend, shading model, full worked example, gotchas. Read this first.
